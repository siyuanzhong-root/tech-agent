## 3. 上下文记忆机制

```
┌─────────────────────────────────────────────────────────────────┐
│                        上下文管理流程                             │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  每次请求处理流程：                                               │
│                                                                 │
│  1. 读取上下文 (getContext)                                       │
│     ├─> 从 UserProfile 读取：                                    │
│     │   - context_studentName (上次查询的学生)                    │
│     │   - context_subject (上次查询的科目)                        │
│     │   - context_semester (上次查询的学期)                       │
│     │   - context_studentId (上次查询的学生ID)                    │
│     │                                                            │
│  2. DeepSeek 信息提取                                            │
│     ├─> 将上下文信息传递给 DeepSeek                               │
│     │   "用户之前查询过学生: 李杰"                                 │
│     │   "用户之前查询过科目: 数学"                                 │
│     │                                                            │
│  3. 智能补全                                                     │
│     ├─> 如果用户说"从高到低排序"                                  │
│     │   DeepSeek 会结合上下文识别为"把李杰的成绩从高到低排序"        │
│     │                                                            │
│  4. 更新上下文 (updateContext)                                    │
│     └─> 将新的查询条件保存到 UserProfile                          │
│         - context_studentName = 李杰                              │
│         - context_subject = 数学                                  │
│         - ...                                                    │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 5. 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端 | HTML5 + CSS3 + JavaScript | 响应式界面 |
| 后端框架 | Spring Boot 2.7.15 | Web服务框架 |
| 数据访问 | Spring Data JPA | ORM框架 |
| 数据库 | SQLite | 轻量级关系型数据库 |
| AI服务 | DeepSeek API | 大语言模型API |
| HTTP客户端 | OkHttp 4.12.0 | API调用 |
| JSON处理 | Gson 2.10.1 | 数据序列化 |
| 构建工具 | Maven | 项目构建 |
